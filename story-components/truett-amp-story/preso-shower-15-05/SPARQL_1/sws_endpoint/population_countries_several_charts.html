The 10 largest countries by population

google.visualization.BarChart

The element which draws the above chart:

<div id="sgvzl_example_query" 
data-sgvizler-endpoint="http://sws.ifi.uio.no/sparql/world" 
data-sgvizler-query="
  SELECT * 
  WHERE { [] a w:Country; w:hasName ?Name; w:hasCountryPopulation ?Population; }
  ORDER BY DESC(?Population) LIMIT 10" 
data-sgvizler-chart="google.visualization.BarChart" 
data-sgvizler-loglevel="2" 
style="width:800px; height:400px;"/>
