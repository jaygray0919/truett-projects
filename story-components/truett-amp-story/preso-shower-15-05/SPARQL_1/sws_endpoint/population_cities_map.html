Ten largest cities by population on a map

google.visualization.Map

The element which draws the above chart:

<div id="sgvzl_example_query" 
data-sgvizler-endpoint="http://sws.ifi.uio.no/sparql/world" 
data-sgvizler-query="
  SELECT ?Name ?Name
  WHERE { [] a w:City; w:hasName ?Name; w:hasCityPopulation ?Population; }
  ORDER BY DESC(?Population) LIMIT 10" 
data-sgvizler-chart="google.visualization.Map" 
data-sgvizler-chart-options="showTip=true" 
data-sgvizler-loglevel="2" 
style="width:800px; height:400px;"/>
