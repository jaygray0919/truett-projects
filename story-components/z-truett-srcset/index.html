srcset Project

Objective

  1. Build an image library for use with srcset specifications
  2. srcset defines the set of images, each with specific size, that a browser will display based on the device.
  3. Mozilla documents a repeatable process here.

Requirements

  1. Read technical process below
  2. Check reference material as needed (page bottom)
  3. Provide cost for creating a srcset-collection from source images (where source image is in one of the following formats: base64, .png, .jpg)

Technical Process

    Identification

  1. Identify base64 font by name and (if possible) by version

    2. Conversion

  2. convert base64 image to webp
  3. convert PNG/JPG image to webp
  4. create srcset for webp items

    5. Optional tools

  5. for BASE64/PNG/JPG-to-webp use:
    • https://cloudconvert.com/
    • https://convertio.co/
    • https://base64.guru/converter/decode/image/png
  6. for webp-to-srcset use:
    • https://www.responsivebreakpoints.com/
    • https://cloudinary.com/documentation/upload_images
