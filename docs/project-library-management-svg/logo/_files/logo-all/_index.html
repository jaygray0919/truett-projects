  • Pronto (symbol)

    Ontomatica's identity specification application is named Pronto, an acronym for Practical ontologies for building electronic labels. Pronto has three components. Substance Data Kit is pre-defined substance descriptions. Each substance (each item) is described using Substance Markup Language - Linked Data (SUML-LD) based on ISO 11238 standard. Product Data Kit is pre-defined product descriptions. Each product (each item) is described using a specific Product Markup Language - Linked Data (PRML-LD) based on ISO 11238 structurally diverse substances standard for defining products. Linked Data Indexing is a service to index chemicals, foods, drugs, cosmetics and textiles based on ISO 11238 Group 1 and Group 4 specifications to describe chemical, food, drug, cosmetic and textile substances, commodities and products.

  • Graph Kits (symbol)

    Ontomatica's knowledge graph application is named Knowledge Graph Kit. Graph Kits are packages of information about food and health products. Packages are readable by humans ("machine-to-person") and are actionable by intelligent devices ("machine-to-machine"). Graph Kits integrate relevant ontology rules (properties, attributes, relationships, constraints) with relevant data sets (items). Each Graph Kit is a starter kit to fast-forward automation of a specific problem.

  • Socrato (symbol)

    Ontomatica's graph analysis application is named Socrato. Socrato's documentation service generates the meaning of objects and vertices in a graph. A visualization service renders a graph in formats such as hierarchy diagrams, phylotrees, state transitions, and parallel coordinates. A reasoning service uses description logic for querying, checking, transforming and filtering information. When data is properly structured, Socrato generates sentences in different natural languages for input to text-to-speech applications.

  • Toronto (symbol)

    Ontomatica's machine automation application is named Toronto, an acronym for torque ontologies. Toronto transfers electronic label information to a machine controller. An example is composition and formulation instructions for an agricultural chemical as defined on a material safety data sheet. When agricultural equipment integates environmental sensors, Toronto-defined chemical components are reformulated in real-time and directed appropriately.

  • Fast Forward (symbol)

    Ontomatica's knowledge transition application is named Fast Forward. Fast Forward extracts unstructured information from existing content and re-organizes content as a knowledge graph. For example, a Fast Forward template extracts recipe information from existing recipes and generates an equivalent knowledge graph. Other Fast Forward templates do similar work for loosely structured instructions.

  • Biograph (symbol)

    Biograph is your personal record of foods and drugs consumed over time. It also is a record of chemicals in cosmetics and personal care products used over time. Biograph is an electronic complement to your electronic health records. Intake and exposure data helps you and your advisers (doctors, trainers, nutritionists) adjust your lifestyle to improve health outcomes. Examples: minimize prediabetes; control type 2 diabetes; maximize microbiome (gut) health; reduce exposure to potentially harmful chemicals; increase consumption of phytochemicals. Unlike a conventional application, you own and control your data. Your data is stored in a secure place and you decide which people and which apps can access your data.
