
.. describe:: amp-list and amp-mustache for persona

   substitution of specific value from a list of values


.. attention::
   depends on <amp-list> and <amp-mustache>.


sample code
-----------

.. highlight:: html

::

   <!-- @format -->
   <amp-story-page id="item-11" data-item-label="Title Page">
   
     <!-- first declaration - logo -->
     <amp-story-grid-layer class="logo-layer" template="vertical">
       <div class="logo-wrapper">
         <amp-list
           template="mst-logo"
           width="auto"
           height="96"
           layout="fixed-height"
           src="amp-state:localState-logo"
         >
           <template id="mst-logo" type="amp-mustache">
             <amp-img
               src="{{logo}}"
               layout="responsive"
               width="80"
               height="80"
             ></amp-img>
           </template>
           <amp-state id="localState-logo">
             <script type="application/json">
               {
                 "items": [
                   {
                     "logo": "https://afdsi.com/static/logo/onto_sym_white_black_back_w421_h421_.png"
                   }
                 ]
               }
             </script>
           </amp-state>
           <div overflow class="list-overflow"></div>
         </amp-list>
       </div>
     </amp-story-grid-layer>

   </amp-story-page>
